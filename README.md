# Quality Assurance v3

Este repositório contém os roteiros de testes **end-to-end (E2E)** da versão 3 da plataforma Affiliates.  
O objetivo é garantir que todas as funcionalidades estejam validadas de forma imparcial, cobrindo os três perfis de acesso disponíveis.

---

## 📂 Estrutura dos Roteiros

- [brands.md](./brands.md) → Testes para o perfil **Marca**  
- [affiliates.md](./affiliates.md) → Testes para o perfil **Afiliado**  
- [accountant.md](./accountant.md) → Testes para o perfil **Financeiro**

Cada arquivo contém:
- Identificação do caso de teste (ID e título)  
- Pré-condições necessárias  
- Passos detalhados para execução  
- Resultado esperado  

---

## 🛠️ Como usar
1. Abra o arquivo correspondente ao teste que deseja realizar.  
2. Siga os passos descritos em cada caso de teste.  
3. Caso encontre falhas, registre como *Issue* neste repositório, vinculando ao ID do caso de teste.  

---

## 🎯 Objetivo
- Garantir cobertura completa dos fluxos críticos da plataforma.  
- Evitar vieses de teste por parte da equipe de desenvolvimento.  
- Centralizar a documentação de QA em um único local acessível.  

---

## 📌 Observações
- Este repositório é dedicado exclusivamente à fase de testes da versão 3.  
- Atualizações e novos casos de teste devem ser adicionados nos arquivos correspondentes.  
- Em caso de dúvidas, entre em contato com o time de QA.
